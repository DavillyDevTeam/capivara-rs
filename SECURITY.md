# Security Policy

## Supported Versions

Security fixes target the **latest release** (when any exist) and the **`main`** branch.
This project is under active construction and not production-released yet.

## Reporting a Vulnerability

Please report vulnerabilities through **GitHub Private Vulnerability Reporting**:

https://github.com/DavillyDevTeam/capivara-rs/security/advisories/new

Include:

- Description and potential impact
- Reproduction steps or proof of concept
- Affected commit or version if known

We will acknowledge reports as soon as practical and keep you informed of progress.

**Do not** open a public GitHub issue for security vulnerabilities.
